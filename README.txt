PROGRAMA PARA PRACTICAR SUMAS PARA NIÑOS PEQUEÑOS

Descripción: El programa permite a niños de primaria practicar sumas, se divide en tres niveles: 

- Nivel Fácil
- Nivel Medio
- Nivel Difícil

Lenguaje de programación: C 

Instalación: 
1. Descargar los archivos
2. Abrir el archivo main en Dev C++
